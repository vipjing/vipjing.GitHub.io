TiDB User Group

TiDB User Group（简称 TUG）是由 TiDB 用户发起的独立用户社区，以线上的 AskTUG 问答社区和线下的 TUG 技术沙龙为大本营。 用户在 TUG 社区可以学习前沿技术知识，发表技术见解，结识圈内朋友，共同建设 TiDB 项目。 目前全国共有北京、上海、杭州、华南（以深圳为中心）和西南（以成都为中心）五个 TUG 区域。TUG 核心成员由以下角色构成：Leader、Co-Leader、Ambassador。
